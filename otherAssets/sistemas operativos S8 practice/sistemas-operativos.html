Actividad S8-A1 - VirtualBox y distribución GNU/Linux de escritorio

1.1 Objetivos

  • Descargar y configurar el software VirtualBox.
  • Instalar un sistema operativo GNU/Linux en su versión de escritorio (desktop).
  • Probar las GUI disponibles para la distribución elegida (Lubuntu).

1.2 Software necesario

  • Oracle VirtualBox
  • Distribución de GNU/Linux en formato ISO (Lubuntu 24.04.2)

1.3 Consigna de la actividad

Requisitos del sistema

Según la información del sistema obtenida en el diagnóstico de DirectX:

  • Sistema operativo: Windows 11 Home Single Language 64 bits
  • Procesador: Intel Core i3-12338 (8 CPUs), ~1.2GHz
  • Memoria RAM: 8GB
  • Espacio en disco: 750GB disponibles

Paso 1: Descargar e instalar VirtualBox

1. Se accedió al sitio oficial de VirtualBox y se descargó la versión compatible con Windows.

2. Se instaló VirtualBox con la configuración predeterminada.

Descargar una imagen ISO de Linux (Lubuntu)

1. Se seleccionó Lubuntu 24.04.2 (Noble Numbat) como distribución de Linux para instalar.

2. Se descargó la imagen ISO de 64-bit para PC (AMD64) desde el sitio oficial.

Página de descarga de Lubuntu mostrando las opciones disponibles

Nota: Lubuntu fue seleccionado por ser una distribución ligera ideal para máquinas virtuales, con requisitos mínimos de 1024MB de RAM.

Crear la máquina virtual

1. En VirtualBox, se hizo clic en "Nueva" para crear una nueva máquina virtual.

2. Se asignó el nombre "lubuntu" a la máquina virtual.

3. Se seleccionó el tipo de sistema operativo como Linux y la versión Ubuntu (64-bit).

Configurar la máquina virtual

1. Se asignaron 2048 MB de memoria RAM (dentro del rango recomendado).

2. Se configuraron 3 CPUs para la máquina virtual.

3. Se creó un disco duro virtual de 25 GB con tamaño dinámico.

4. En la pestaña Almacenamiento, se seleccionó la imagen ISO descargada como medio de instalación.

1. Se inició la máquina virtual haciendo clic en "Iniciar".

2. En el menú de inicio, se seleccionó "Install Lubuntu".

Resultado: La instalación de Lubuntu en VirtualBox se completó exitosamente. El sistema operativo funciona correctamente con la configuración asignada, permitiendo explorar todas las características de esta distribución ligera de Linux.

Conclusión

La actividad permitió familiarizarse con el proceso de creación de máquinas virtuales usando VirtualBox y la instalación de sistemas operativos GNU/Linux. Lubuntu 24.04.2 demostró ser una excelente opción para máquinas virtuales gracias a sus bajos requisitos de sistema y su interfaz gráfica ligera pero funcional.

El proceso completo, desde la descarga de los componentes necesarios hasta la instalación final del sistema operativo, se documentó en este informe con capturas de pantalla que muestran cada paso importante.
